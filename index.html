html

El html es un lenguaje que se utiliza para crear paginas en internet. El Lenguaje de Marcado de Hipertexto.
Esta compuesto por varias estiquetas que el navegador interpreta y da forma en la pantalla. Sus etiquetas
se utilizan para poner:Hipervinculos, Textos, Tablas, Listas, entre otros.


Las principales etiquetas de html son:

Body: Para el contenido.
Head: Para informacion del documento.
Div: Mover cosas en la pantalla.
a: Hipervinculos
br: Saltos en linea.
Img: Para imagenes




¿Como crear un link en Github?

Creamos una cuenta en Github y darle en la opcion de crear un nuevo repositorio, despues le ponemos un nombre y elegimos si nuesttro archivo va a ser Publico o Privado, luego cuando queramos subir algun trabajo seleccionamos "subir un archivo existente" TODOS los archivos de la carpeta que contiene el trabajo y lo ponemos en un espacio grande que se muestra en github, despues le damos en la opcion de "main" y escribimos gh.pages, cuando se haya creado el "branch" vamos a settings buscamos branches vamos a las flechas de un cuadro que tiene el nombre de main y seleccionamos gh-pages- uptade- I understand y luego vamos a options bajamos hasta encontrar nuestro link de Github.




Como editar un archivo desde Github

Cuando queramos editar un archivo de Github los pasos son sencillos, cuando entramos a nuestra cuenta de Github en la pagina principal nos van a aparecer los archivos de la carpeta que subimos al crear el link, le damos en el archivo que queremos editar y nos aparecera el algoritmo del archivo vamos a la esquina de la parte superior de donde esta el algoritmo y le damos en el lapiz que dira: "edit this file" luego que le demos ahi aparecera el algoritmo listo para ser editado.




Contenedores

Guardamos el archivo en donde realizaremos el contenedor, ponemos la etiqueta div despues ponemos el modificador de div Style y empezamos a escribir sus modificadores los cuales son: position, absolute y cuadramos la posicion en donde queremos que este nuestro con tenedor con las etiquetas: top, left o rigth, luego ponemos la etiqueta width que es para su tamaño la etiqueta height para su alto y background para su fondo.
